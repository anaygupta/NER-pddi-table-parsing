<td colspan="7" stylecode="Rrule Botrule Lrule " valign="middle">
<paragraph>S = Susceptible, I = Intermediate, R = Resistant </paragraph>
<paragraph>No interpretative criteria have been established for testing enterococci. </paragraph>
<paragraph>Susceptibility of staphylococci to meropenem may be deduced from testing penicillin and either cefoxitin or oxacillin.</paragraph>
<paragraph>
<sup>a </sup>The current absence of data on resistant isolates precludes defining any category other than “Susceptible”. If isolates yield MIC results other than susceptible, they should be submitted to a reference laboratory for additional testing.</paragraph>
<paragraph>
<sup>b </sup>Interpretative criteria in Table 2 should only be used for isolates from CSF. For nonmeningitis isolates of <content stylecode="emphasis">Streptococcus pneumoniae</content> a penicillin MIC of ≤ 0.06 mcg/mL or oxacillin zone ≥ 20 mm can predict susceptibility to meropenem. MIC testing should be performed on isolates that do not test as susceptible by either of these methods, and on all meningitis <content stylecode="emphasis">S. pneumoniae</content> isolates.</paragraph>
<paragraph>
<sup>c</sup> Viridans group streptococci should be tested for meropenem susceptibility using an MIC method</paragraph>
<paragraph>
<sup>d</sup> Reliable disk diffusion tests for meropenem do not yet exist for testing streptococci</paragraph>
<paragraph>
<sup>e</sup> MIC values using either Brucella blood or Wilkins Chalgren agar (former reference medium) are considered equivalent, based upon published <content stylecode="emphasis">in vitro</content> literature and a multicenter collaborative trial for these antimicrobial agents.</paragraph>
